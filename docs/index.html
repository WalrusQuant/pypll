pypll

A Python client for the Premier Lacrosse League stats API. Every function returns a pandas DataFrame.

Get Started Function Reference
$ pip install pypll
Getting Started

Installation

Install pypll from PyPI with a single command. Python 3.8+ and pandas are required.

From PyPI

bash 
pip install pypll

Development Install

bash 
# Install from source
pip install -e .
Usage

Quick Start

Import pypll and start pulling data in seconds. All functions return pandas DataFrames ready for analysis.

python 
import pypll

# Team standings
standings = pypll.pll_standings(year=2025)
print(standings[["fullName", "wins", "losses", "scoreDiff"]])

# Player season stats (~70 columns)
stats = pypll.pll_player_stats(year=2025, season_segment="champseries")
print(stats[["firstName", "lastName", "position", "goals", "assists"]].head())

# Game schedule and results
events = pypll.pll_events(year=2025, include_cs=True, include_wll=False)
print(events[["startTime", "home_fullName", "away_fullName", "homeScore", "awayScore"]].head())

# Official draft order
draft = pypll.pll_draft_order(year=2025, league="PLL")
print(draft[["round", "overallPick", "teamId", "pick_playerName"]].head())

# Mock draft predictions from analysts
predictions = pypll.pll_draft_predictions(year=2026)
print(predictions[["analystName", "playerName", "position", "overallRank"]].head())

# Free agent signings
free_agents = pypll.pll_free_agents(year=2026)
print(free_agents[["name", "position", "prevTeamId", "newTeamId", "newContractStatus"]].head())

API

Function Reference

All functions are available directly on the pypll module. Each returns a pd.DataFrame.

pll_standings (year=2025, champ_series=True)

Fetch team standings for a given season. Returns one row per team.

Parameter Type Default Description
year int 2025 Season year
champ_series bool True True for championship series standings, False for regular season
Returns: pd.DataFrame — one row per team
Key Columns
teamId fullName location wins losses ties scores scoresAgainst scoreDiff conference conferenceSeed seed
pll_player_stats (year=2025, season_segment="champseries")

Fetch player season statistics. Returns one row per player with approximately 70 columns spanning identity, team info, and stats.

Parameter Type Default Description
year int 2025 Season year
season_segment str "champseries" Segment of the season: "champseries" (playoffs) or "regular"
Returns: pd.DataFrame — one row per player, ~70 columns
Column Groups
Player Identity
officialId firstName lastName slug profileUrl jerseyNum position positionName experience
Team Info (team_ prefix)
team_teamId team_fullName team_location ...
Stats (50+ columns)
goals assists shots groundBalls causedTurnovers faceoffWins saves ...
pll_events (year=2025, include_cs=True, include_wll=True)

Fetch the game schedule and results. Returns one row per game event, including scores, venue, broadcaster, and prefixed team info for both home and away sides.

Parameter Type Default Description
year int 2025 Season year
include_cs bool True Include championship series (playoff) games
include_wll bool True Include Women's Lacrosse League (WLL) games
Returns: pd.DataFrame — one row per game event
Key Columns
eventId startTime status homeScore awayScore venue broadcaster league home_* away_*
pll_draft_order (year=2025, league="PLL")

Fetch the official draft order. Returns one row per draft pick. Player fields are None for picks with no player on record.

Parameter Type Default Description
year int 2025 Draft year
league str "PLL" League identifier: "PLL" or "WLL"
Returns: pd.DataFrame — one row per draft pick
Key Columns
id year round roundPick overallPick teamId league pick_playerName pick_college pick_position pick_playerId
pll_draft_predictions (year=2026)

Fetch mock draft predictions published by PLL analysts. Returns one row per analyst prediction. Typically published in the months leading up to the draft.

Parameter Type Default Description
year int 2026 Draft year
Returns: pd.DataFrame — one row per analyst prediction
Key Columns
analystName playerName position college overallRank positionRank change analysis league
pll_free_agents (year=2026)

Fetch free agent signings and available players. Returns one row per free agent. Data becomes available once the free agency period opens.

Parameter Type Default Description
year int 2026 Free agency year
Returns: pd.DataFrame — one row per free agent
Key Columns
name firstName lastName position age experience prevTeamId newTeamId newContractStatus thirtyPercentThresholdMet
Notes

Data Availability

Not all data is available at all times. Here's what to expect across the calendar year.

Season data (standings, player stats, events) is only available during or after the relevant season has begun. Querying a future year or a year before the season starts will return an empty DataFrame or raise a RuntimeError.

Draft order data is available once the draft has taken place. Mock draft predictions are typically published in the months leading up to the draft. Free agent data becomes available once the free agency period opens.
